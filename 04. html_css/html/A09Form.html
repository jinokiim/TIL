Form

사용자로부터 어떠한 값을 받을 목적으로 기술
전달될 데이터는 form 태그에 포함되어야 한다.
GET: 256Byte - 4000Byte 이내의 크기의 데이터만 전송 가능
Post: 크기 상관없음
body 처음에 붙어서 넘어간다. (내용을 볼 수 없음)

상품 주문
size는 필드의 크기(입력할 수 잇는 칸의 크기)
maxlength 정의한 필드에 입력할 수 있는 최대 크기
required: 값이 반드시 입력되어야 한다.
readonly: 읽기 전용. 사용자가 값을 입력할 수 없음.
disable: 사용하지 않음. 사용자가 값을 입력할 수 없음. 지정한 action에 값을 전달하지 않음
주문한 사람



이 히든필드는 화면에 표시되지 않음.
사용자는 모르게 서버에 어떠한 값을 전송할 목적으로 사용한다.
배송지







radio, checkbox를 하나 묶은 형태(공간을 적게 차지한다)
select value 속성이 없으면 option에 적은 문자열이 전송된다.
